How it works

The demo alternates two systems: the LLM handles language and meaning (reading the note, translating and normalizing each term, flagging negation), while the SNOMED CT terminology server (Snowstorm) finds and confirms the concept. That back-and-forth is the core idea.

No confident match? The LLM offers a broader or alternative term and Snowstorm searches again — LLM ↔ Snowstorm — until a concept is accepted or the entity is left unresolved. Open any row's Flow to see the exact steps for an entity.

A simplified re-implementation inspired by IHTSDO/llm-chain-entity-extraction; see the 2023 SNOMED Expo poster.
